# A Simple Todo List -- Vue.js Version
>
> Powered By HTML,CSS and Vue.js <br> Also made use of Normalize.css

## Features
+ Add todo item
+ Remove todo item
+ Finish todo item
+ Local storage
+ Display by category

## Quick Start

npm install

npm run dev
